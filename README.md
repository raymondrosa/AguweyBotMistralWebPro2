# 🤖 AguweyBot - Asistente con Ministral-3

Asistente inteligente con análisis de documentos, audio y guardado de conversaciones usando Mistral AI.

## 🚀 Características

- 📄 **Análisis de documentos**: PDF, Excel, CSV, TXT, DOCX
- 🔊 **Texto a voz**: Escucha las respuestas
- 💾 **Guardado de conversaciones**: Almacena y carga historiales
- 📋 **Copiado rápido**: Botón para copiar respuestas
- 🎨 **Interfaz moderna**: Diseño con tema cyberpunk/neón

## 🛠️ Instalación Local

```bash
# Clonar repositorio
git clone https://github.com/tu-usuario/aguweybot.git
cd aguweybot

# Instalar dependencias
pip install -r requirements.txt

# Ejecutar
streamlit run aguweybotweb.py
